# Model Weave 0.1.9

Model Weave 0.1.9 では、Mermaid renderer の表現力強化と、`app_process` / Business Flow の記述・表示改善を中心に更新しました。

## Highlights

### Mermaid Detail view を追加

Class / ER 系モデルで、従来の軽量 Mermaid view に加えて、より詳細な Mermaid Detail view を追加しました。

従来の Mermaid view は全体俯瞰向けとして維持し、Mermaid Detail view では、属性・型・PK/FK・リレーションなど、レビューや共有に使いやすい情報量を表示します。

対象:

* Class
* Class Diagram
* Entity
* ER Diagram

Custom renderer は引き続き、詳細レビュー、診断、ナビゲーション向けの表示として利用できます。

### Mermaid Source panel を追加

Mermaid 系 renderer で、生成された Mermaid source を確認できる折りたたみパネルを追加しました。

生成された Mermaid code block を通常の Obsidian Markdown に貼り付けることで、Model Weave 外の文書でも図として再利用しやすくなりました。

### Mermaid Debug / Source 表示を整理

Mermaid Source と Mermaid Render Debug の表示位置と扱いを整理しました。

* Mermaid Source は Mermaid renderer で利用可能
* Mermaid Render Debug は設定で表示を切り替え
* Debug / Source の配置を Mermaid 系 view 間で統一

### Business Flow 表示を改善

`app_process` の Business Flow preview を改善しました。

主な変更:

* `Steps.kind` に応じたノード形状の描き分け
* `Steps` の行順からの暗黙フロー生成
* `Flows` による分岐・合流・ループ・例外経路の明示
* `Flows.label` と `Flows.condition` の表示優先順位整理
* Wikilink を含む condition 表示の改善
* Business Flow のタイトル位置を他の図ビューと整合

`Steps` に主な処理順序を書き、`Flows` には分岐や合流など明示したい接続だけを書く、という使い方がしやすくなりました。

### Render mode の仕様を整理

表示モードの初期選択ルールを整理しました。

* frontmatter の `render_mode` を最優先
* 未指定の場合はフォーマット別の初期表示設定を使用
* ツールバー上の renderer 切り替えは現在のビューだけの一時変更
* `auto` は通常のユーザー向け初期表示モードとしては扱わない方針へ整理
* 対応していない `render_mode` は warning を表示して安全に fallback

### PNG export のファイル名を改善

PNG export 時のファイル名に renderer 名を含めるようにしました。

これにより、同じモデルを Custom / Mermaid / Mermaid Detail など複数の renderer で出力しても、ファイルが上書きされにくくなりました。

例:

* `SampleClassDiagram__custom.png`
* `SampleClassDiagram__mermaid.png`
* `SampleClassDiagram__mermaid-detail.png`

## Documentation

0.1.9 では、関連ドキュメントも更新しました。

主な更新:

* `app_process` format の `Steps` / `Flows` 仕様を明確化
* Business Flow の暗黙フローと明示フローの組み合わせを説明
* `Steps.kind` による図形の違いを説明
* `render_mode` の最新方針を反映
* 古い V0.7 rendering policy を V0.8 rendering policy に置き換え
* 日本語版 / 英語版の `FORMAT-app_process` を整合

## Notes

Mermaid の kind / area / layer による色分けは、0.1.9 では正式機能としては導入していません。

将来的には、Mermaid / Custom renderer / ERD / Application Flow などに共通して適用できる Color Scheme 機能として検討します。

また、Sequence Diagram や State Transition Diagram は、0.2.0 以降の機能候補として整理しています。
