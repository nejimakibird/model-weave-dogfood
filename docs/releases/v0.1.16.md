# Model Weave 0.1.16

Model Weave 0.1.16 focuses on Viewer / Preview usability, diagram navigation, release compatibility cleanup, and lint readiness.

## Compatibility

Starting with 0.1.16, Model Weave requires Obsidian 1.8.7 or later. Earlier released versions keep their original minimum Obsidian version entries.

## Highlights

* Improved the Viewer with Focus mode, View Only mode, viewport state handling, graph titles, and more consistent graph toolbar placement.
* Reorganized the lower Review Panel with Review Summary, Diagnostics, Relationships / Impact, Source Links, Weave Map, Details, and Mermaid Source sections.
* Added grouped Relationships / Impact summaries and Markdown-friendly copy output for relationship reviews.
* Added Source Links summary and bulk copy actions for all paths, available paths, missing paths, and Markdown lists.
* Improved Diagnostics cards with clearer metadata, Open location, and copy actions.
* Added hover preview and click navigation for resolved nodes and boxes in supported Model Weave views, including Mermaid-based and custom-rendered views.
* Added commands to open the current preview in the main pane or a new pane.

## Viewer interaction notes

Resolved nodes and boxes in supported Model Weave views can show hover previews and can be clicked to open their source Markdown. Unresolved nodes, labels, edges, and field rows may not be interactive.

Hover previews rely on Obsidian's hover-link / Page Preview behavior. They may not appear near the top edge of the Obsidian window or while using Model Weave Focus mode. Use click navigation as the reliable fallback for resolved nodes.
