# DefaultColorScheme

## Summary

Default Color Scheme for Model Weave sample diagrams.

Set `defaultColorSchemeRef` to `[[COLOR-SCHEME-DEFAULT]]` in Model Weave settings to preview colors in Domains, Domain Diagram, DFD, and app_process Business Flow views.
